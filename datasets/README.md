# 📊 Datasets Information

This directory contains instructions and metadata for the downstream fine-tuning datasets. **Download Link**: [Quark Drive Link](https://pan.quark.cn/s/662574b2c536)

When running the fine-tuning script (`src/fine-tune.py`), you must correctly specify the number of classes for the targeted dataset using the `--nb_classes <num-class>` argument. 

Please refer to the table below for the precise `<num-class>` associated with each dataset:

| Dataset Name              | `<num-class>` (Number of Classes) |
| :------------------------ | :-------------------------------: |
| **CrossPlatform-Android** |               `181`               |
| **CrossPlatform-iOS**     |               `124`               |
| **CICIoT2022**            |                `6`                |
| **ISCXTor2016**           |                `8`                |
| **ISCXVPN2016**           |                `8`                |
| **USTC-TFC2016**          |               `20`                |
| **cstnet-tls_1.3**        |               `119`               |
